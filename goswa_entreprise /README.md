# Goswa Entreprise

Système d'entreprise pour FiveM avec tablette pour patrons et admins.

## Fonctionnalités

- F6 pour toggle service des employés
- Tablette patron/admin avec liste des employés
- Ajouter des heures manuellement
- Accès tablette limité aux patrons et admins

## Installation

1. Copier le dossier `goswa_entreprise` dans votre répertoire `resources` de FiveM.
2. Ajouter `start goswa_entreprise` dans votre `server.cfg`.
3. Installer `ESX` et `mysql-async` si nécessaire.
